Setting up plugins is straightforward, simply download and paste the file into the right folder

## Installation

<Stepper>
    <StepperItem title="Step 1: Navigate to the Plugin Page">
        Use the search bar on the top of the page to find your desired plugin
    </StepperItem>
    <StepperItem title="Step 2: Download the Plugin">
        Download the plugin by visiting the plugin's download page
    </StepperItem>
    <StepperItem title="Step 3: Install the Plugin">
        Once downloaded, simply paste the plugin file into the `plugins` folder in your Minecraft server files
    </StepperItem>
</Stepper>
