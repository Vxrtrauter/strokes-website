Setting up mods is a little tricky, download and paste the file into the right folder

## Installation

<Stepper>
    <StepperItem title="Step 1: Navigate to the Mod Page">
        Use the search bar on the top of the page to find your desired mod
    </StepperItem>
    <StepperItem title="Step 2: Download the Mod">
        Download the plugin by visiting the Mod's download page
        <Note title="Warning" type="warning"> 
            Please note that you have to pick the correct version of the mod which is compatible with your Minecraft version otherwise it will not work
        </Note>
    </StepperItem>
    <StepperItem title="Step 3: Install the Mod">
        Once downloaded, simply paste the mod file into the `mods` folder in your Minecraft game files
    </StepperItem>
</Stepper>
